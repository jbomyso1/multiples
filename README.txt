Program intended to take user input of an integer, and will output the list of multiples of 3 and 5 up to that value.  The program will also sum the values and output a total sum.  

Repeats of multiples (15, 30, 45, etc) will not count double towards the list or the sum.

User Instructions:
Program Execution: ./mult35

When prompted, enter desired integer value and press enter.
